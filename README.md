# Codveda_technologies-task-2
Task 2: Exploratory Data Analysis (EDA)

This project focuses on understanding and exploring the Iris dataset through statistical analysis and visual exploration.

Objectives
Analyze dataset structure and descriptive statistics
Study feature distributions
Detect outliers
Analyze relationships between variables
Identify correlations among numerical features

Techniques Used
Histograms,
Boxplots,
Scatter Plots,
Correlation Matrix,
Summary Statistics

Tools Used
Python,
Pandas,
Matplotlib,
Seaborn

Outcome
Generated meaningful insights about data distribution, feature relationships, and correlations that can support future predictive modeling.
